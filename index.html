Abstract

ShadowGen turns a single product image into a shadow-aware composed output. The current system separates orchestration from heavy inference backends: detection, segmentation, depth, normals, shadow synthesis, and composition can run locally, through mock adapters, or through Triton-ready stage boundaries. This page is structured like a standard academic model-release page so that video, method notes, results, repositories, and citation details can be filled in as the model matures.

Highlights

The release story is framed around practical image production: controllable shadows, inspectable stages, and serving-friendly model boundaries.
